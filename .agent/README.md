# .agent 공유 문서 폴더

이 폴더는 에이전트 간 공식 전달 문서 위치입니다.

## 기본 파일

- `STATUS.md`: 전체 진행상태/블로커
- `API_SPEC.md`: 백엔드 API 명세
- `DB_REQUEST.md`: 백엔드가 DBA에 전달하는 DB 요청
- `DB_SPEC.md`: DBA 스키마/운영 문서
- `DB_HANDOFF_TO_BACKEND.md`: DBA -> Backend 전달
- `FRONTEND_INTEGRATION.md`: 프론트 API 연동 명세

## 규칙

- 텔레그램 대화보다 문서가 기준입니다.
- 파일 생성/수정 시 마지막 업데이트 시간을 남기세요.
- 블로커는 `STATUS.md`에 즉시 기록하세요.
